GBS Agincourt

To run GBS Agincourt properly, install Fingerprint, Afis, ICAO Face and WSQ SDK licenses at c:\ProgramData\Griaule.
To retrieve Griaule licences, contact us at www.griaulebiometrics.com

Install sigplus.exe found on bin/sigplus to properly load Topaz libraries.

To change access to GBS Server after instalation, edit the file GBSAgincourt.properties located at C:\Program Files (x86)\Griaule\GBSAgincourt\conf.
There, change the properties:

server.url
server.applicationGuid
server.accountGuid
server.authenticationKey

According to your GBS Server. Or, inside the Agincourt, go to menu Configuration\Configuration and change setting there.
